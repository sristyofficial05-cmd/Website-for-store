Mithila Store
Preserving Culture. Empowering Artisans
Mithila Store is an e-commerce platform dedicated to showcasing and selling authentic Mithila (Madhubani) art and products.
Our mission is to digitally empower local artisans and bring the rich cultural heritage of Bihar to a global audience.
Features
🖼️ Browse traditional Mithila paintings and crafts
🛒 Add to cart & secure checkout system
👤 User authentication (Login / Register)
📦 Order management system
🔍 Search and filter products
📱 Responsive UI (mobile-friendly)
🧑‍🎨 Artisan-focused product listings.
